#  Linux Labs - Administración y Operaciones

Repositorio dedicado a la documentación de laboratorios prácticos sobre administración de sistemas Linux, orientado a perfiles de **Junior Sysadmin, NOC y Cloud Security**.

##  Objetivo
El propósito de este directorio es demostrar capacidad técnica en:
- Configuración y endurecimiento (hardening) de sistemas Linux.
- Gestión eficiente de usuarios, grupos y políticas de permisos.
- Automatización de tareas rutinarias y gestión de servicios.
- Resolución de incidencias en entornos de servidor.

##  Entorno de Trabajo
- **Distribución:** Ubuntu Server / Debian (basado en el stack de NDG Linux Essentials).
- **Entorno:** Home Lab local sobre **VirtualBox**.
- **Acceso:** Gestión remota vía **SSH** y administración por línea de comandos (CLI).
