# IT Labs – Home Lab de Operaciones IT

Repositorio personal de prácticas de administración de sistemas Linux,
enfocado a tareas reales de operaciones IT (junior sysadmin / NOC).

## Objetivo
Este repositorio sirve como evidencia práctica de:
- Administración básica de sistemas Linux
- Gestión de usuarios y permisos
- Buenas prácticas operativas y documentación

## Estado actual
- Formación en curso: NDG Linux Essentials
- Tipo de laboratorio: Home lab local
- Nivel: fundamentos de operaciones IT

## Entorno de laboratorio
- Sistema operativo: Linux
- Tipo: máquina local / virtual
- Acceso: consola y SSH

## Contenido previsto
- Gestión de usuarios y grupos
- Permisos de archivos y directorios
- Procesos y servicios básicos
- Documentación de tareas operativas
