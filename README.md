# IT Labs – Portfolio de Formación y Práctica

Repositorio personal de aprendizaje y laboratorios prácticos en **IT, sistemas, cloud y ciberseguridad**.  
Enfocado a demostrar competencias reales alineadas con tareas de **operaciones IT / sysadmin / cloud junior / blue team**.

---

##  Objetivo

Este repositorio sirve como **evidencia práctica** de:

- Administración de sistemas Linux y Windows
- Fundamentos de redes y servicios
- Cloud (AWS)
- Seguridad defensiva y ofensiva básica
- Buenas prácticas operativas y documentación técnica
- Capacidad de aprendizaje continuo y autoformación

---

##  Roadmap de formación

Planning personal de 24 meses (6–8h semanales):

### Linux
- NDG Linux Unhatched
- NDG Linux Essentials
- Linux Foundation – Introduction to Linux

### Cloud & Seguridad
- AWS Cloud Practitioner
- AWS Solutions Architect Associate (SAA-C03)
- CompTIA Security+ (SY0-701)

### Defensa
- Blue Team Junior
- BTLO Level 1

### Ofensiva
- Fundamentos de Pentesting
- eJPT / TryHackMe (ruta ofensiva)

---

##  Tipo de laboratorio

- Home Lab local (máquinas virtuales)

- Laboratorios guiados y escenarios prácticos
- Scripts y automatizaciones básicas
